# multiplicador_escolar

## mult_escola_enzo.py
multiply numbers in the school way

ex:  

<pre>
      98765
    x 12345
------------
     493825
   +395060
  +296295
 +197530
 +98765
-----------
 1219253925
</pre>

### signal
get the signal of a number

ex:
<pre>
>>> signal(98765)
1
>>> signal(-98765)
-1
>>> signal(0)
1
</pre>
